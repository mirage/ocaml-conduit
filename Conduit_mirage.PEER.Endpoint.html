type error = [ `Unknown of string ] 
val server : domid:int ->
       port:port ->
       ?read_size:int -> ?write_size:int -> unit -> t Lwt.t
server ~domid ~port ?read_size ?write_size () will listen on a connection for a source domid and port combination, block until a client connects, and then return a Conduit_mirage.ENDPOINT.t handle to read and write on the resulting connection. The size of the shared memory buffer can be controlled by setting read_size or write_size in bytes.
val client : domid:int ->
       port:port -> unit -> t Lwt.t
client ~domid ~port () will connect to a remote domid and port combination, where a server should already be listening after making a call to Conduit_mirage.ENDPOINT.server. The call will block until a connection is established, after which it will return a Conduit_mirage.ENDPOINT.t handle that can be used to read or write on the shared memory connection.
val close : flow -> unit io
Close a Vchan. This deallocates the Vchan and attempts to free its resources. The other side is notified of the close, but can still read any data pending prior to the close.
